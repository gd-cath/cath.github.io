Markdown简介

       Markdown是一种纯文本格式的标记语言。通过简单的标记语法，它可以使普通文本内容有一定的格式。

优点：

  1. 因为是纯文本，所以只要支持Markdown的地方都能获得一样的编辑效果，可以让作者摆脱排版的困扰，专心写作。
  2. 操作简单。比如：WYSIWYG编辑时标记个标题，先选中内容，再点击导航栏的标题按钮，选择几级标题。要三个步骤。而Markdown只需要在标题内容钱加#即可

缺点：

  1. 需要记一些语法（比较简单）
  2. 有些平台不支持Markdown编辑模式。

Markdown语法

标题

      在想要设置为标题的文字前面加#来表示

      一个#是一级标题，二个#是耳机标题，以此类推。支持六级标题。

      注：标准语法一般在#后面跟个空格再写文字。

      示例：

       # 这是一级标题

       ## 这是二级标题

       ### 这是三级标题

       #### 这是四级标题

       ##### 这是五级标题

       ###### 这是六级标题

粗体、斜体、删除线和下划线

      Markdown语法：

      *斜体*

      **粗体**

      ***加粗斜体***

      ~~删除线~~

      Typora快捷键

      ctrl+b:粗体

引用块

      在引用的文字前加>即可。引用也可以嵌套，如加两个>>三个>>>

      Markdown语法:

       >这是引用的内容

       >>这是引用的内容

      Typora快捷键:

      ctrl+shift+q

代码块

      Markdown语法:

       `单行代码`

       ```

       多行代码

       ```

分割线

      Markdown语法:

       方法一：——

       方法二: +++

       方法三：***

       使用以上三种方法都可以

列表

      Markdown语法：

      1. 有序列表项

      * 无序列表项

      + 无序列表项

      - 无序列表项

      注意：* + - 1. 跟内容之间都要有一个空格；列表嵌套时，上一级和下一级之间敲三个空格即可
